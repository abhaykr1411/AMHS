# AMHS: (Asset Management & Helpdesk System)

## Prerequisites
* **Node.js** (v14 or higher)
* **MySQL Server**
* **Git**

## 1. Database Setup
1. Open your MySQL terminal or Workbench.
2. Run the SQL script located at `/database/schema.sql` to create the `inventory` database and required tables.
